# TheSixth_ModularFiles
C++. Лаба №6 (6 вариант). Cд, объединения, перечисления.

## №1 - ExamTaskC2 
На вход подаются сведения о клиентах фитнес-центра. В первой строке указывается целое число **N**, а каждая из последующих **N-строк** имеет формат: 
**<Год> <Номер месяца> <Продолжительность занятий (в часах)> <Код клиента>**.<br>
Все данные целочисленные. Значение года лежит в диапазоне **от 2000 до 2010**, код клиента — **в диапазоне 10–99**, продолжительность занятий — **в диапазоне 1–30**. Найти строку исходных данных с максимальной продолжительностью занятий. Вывести эту продолжительность, а также соответствующие ей год и номер месяца (в указанном порядке). Если имеется несколько строк с максимальной продолжительностью, то вывести данные той из них, которая является первой в исходном наборе.

## №2 - 2
Имеется информация об учениках младшей школы:<br>
Для всех учеников известны: **фамилия, имя, класс**;<br>
Для учеников **1-х** классов дополнительно известна их скорость чтения **(слов в минуту, тип short int)**;<br>
Для учеников **4-х** классов известны баллы итоговой аттестации **(единый муниципальный тест от 1 до 100 баллов, тип float)**;<br>
Для учеников **2-х** и **3-х** классов известны данные итоговой школьной контрольной по математике **(оценки от 1 до 10 баллов, тип short int)**.<br>
Выполнить следующее:<br>
Описать структурный тип **«ученик»**. **Обязательно использовать тип union!**;<br>
Написать процедуру, позволяющую ввести с клавиатуры данные для одного ученика;<br>
Используя эту процедуру, ввести сведения об **N-учениках** и сохранить их в текстовом файле;<br>
Распечатать на экране содержимое данного файла в виде таблицы.

## №3 - 3
Работа со студентами. Создать структуру **Student**, которая содержит: **имя студента, возраст, средний балл**.<br>
Реализовать функции для:<br>
Добавления студента;<br>
Вывода информации о студенте;<br>
Подсчета количества студентов с высоким средним баллом **(например, выше 4.0)**;<br>
Поиск студентов по возрасту.<br>
Добавить возможность изменения информации о студенте и вычисления общего среднего балла для определённых студентов.
